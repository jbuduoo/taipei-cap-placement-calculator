# 115 會考基北區落點估算

這是一個單頁式的國中教育會考基北區落點估算工具，主要用來快速試算會考成績級分，並對照台北、新北、基隆區高中與高工/高商的參考落點資料。

## 功能

- 支援五科成績輸入：國文、英文、數學、社會、自然。
- 可用「換算今100分制估算」輸入，也可直接選擇各科等級。
- 依 115 年會考級距表換算 A++、A+、A、B++、B+、B、C。
- 自動計算五科會考積分、等級組合與落點筆數。
- 高中落點列表顯示 113、114、115 預估分數與落點狀態。
- 高工/高商以校科為單位顯示，包含公私立、縣市、區域、學校、科別與歷年分數。
- 高工/高商支援關鍵字搜尋，可查學校、科別、地區，例如資訊、餐飲、開平、板橋。
- 沒有錄取分數但有校科資料的項目，會在搜尋時顯示為「無分數資料」。
- 內建 115 級距表圖，可展開查看各科等級對照。

## 主要檔案

- `outputs/junior-exam-taipei-newtaipei-calculator-sunrise.html`：目前選定的暖色明亮版主檔。
- `outputs/junior-exam-taipei-newtaipei-calculator.html`：原始配色版本。
- `outputs/junior-exam-taipei-newtaipei-calculator-ocean.html`：清爽藍綠版。
- `outputs/junior-exam-taipei-newtaipei-calculator-graphite.html`：深色高對比版。
- `outputs/115-grade-table.png`：115 年會考各科級距表圖片。

## 使用方式

直接用瀏覽器開啟 `outputs/junior-exam-taipei-newtaipei-calculator-sunrise.html` 即可使用，不需要安裝伺服器或額外套件。

## 注意事項

本工具是依已整理資料進行落點估算，僅供參考。實際錄取結果仍會受到當年度招生名額、志願序、多元學習表現、同分比序與各校科招生狀況影響。
